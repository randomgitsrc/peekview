Context-aware

Overflow 内容按权限/文件类型动态展示。Owner 看到 Make Private + Share + Download + Pack + Raw + Delete。Guest 看到 Download + Raw。文件是 .md 时才展示 TOC。

Grouping & hierarchy

Divider 分隔 Owner 操作 / 通用操作 / 危险区域。Delete 单独在底部红色高亮，避免误触。

Desktop overflow dropdown

Desktop 用 More▾ 下拉菜单（同内容），hover 展开。Mobile 用 Bottom Sheet，符合平台习惯，单手可触。

No emoji icons

所有操作使用 Lucide SVG 图标（16-18px），符合 DESIGN.md §2 icon 规范。危险操作红色 `--c-error` 高亮。
