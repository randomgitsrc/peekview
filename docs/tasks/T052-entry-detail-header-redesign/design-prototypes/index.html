Header Redesign v3

完全重新设计。核心改动：Desktop 从 4 行→2 行（78px），按钮用 icon-only + overflow 下拉菜单。
Mobile 底部栏动态适配文件类型，用户+标签移到顶部随滚动隐藏，Files 回到 info 区左侧。
思路来源：GitHub 文件头、Linear issue 详情、Notion 页面栏——都是紧凑 header + overflow 下拉。

s1 — Desktop: 紧凑 2 行 header PRIMARY→

第一行：Logo + Title + [Copy] [Share] [More▾] [🌙]。第二行：meta + tags + 状态，inline。~78px 总高。

s2 — Mobile: 动态底部栏 + FAB MOBILE→

Sticky header + 内容区 title+meta+tags（随滚动隐藏）+ Files 按钮靠左 + 3 种底部栏状态。

s3 — Mobile: Bottom Sheet 抽屉 DRAWER→
