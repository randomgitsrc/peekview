## 🔴 严重问题分析 (9个)

### 问题 1: 滚动功能完全失效

**影响**: 高 - 长内容无法查看，核心功能阻塞  
**根因**: CSS `overflow` 配置错误

| 组件 | 当前代码 | 问题 |
|------|----------|------|
| App.vue | `min-height: 100vh` | 无 `overflow-y: auto`，页面无法滚动 |
| CodeViewer.vue | `overflow-x: auto` | 无 `overflow-y`，代码区无法垂直滚动 |
| MarkdownViewer.vue | 无 overflow | 无滚动配置，长文档无法查看 |
| EntryDetailView.vue | 无 max-height | 内容溢出，无法滚动 |

**修复方案**: 添加 `overflow: auto` 和 `max-height` 限制
