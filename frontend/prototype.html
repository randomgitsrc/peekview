README.md

This is a demonstration of the Peek UI design system. This paragraph demonstrates text wrapping behavior - long text should wrap automatically while code blocks and tables remain horizontally scrollable.

Installation

Install the package using pip:

bash 
pip install peek

Usage

Basic usage example with a very long line that should wrap properly in the paragraph but not in the code block below:

python 
from peek import create_entry

# This is a long comment that demonstrates how the code block handles overflow - it should scroll horizontally rather than wrap

Configuration

Supported configuration options:

Option Type Default Description
host string "127.0.0.1" Server bind address
port number 8080 Server port number that the application will listen on
data_dir string "~/.peek" Directory for storing application data and files

API Reference

The API provides endpoints for entries, files, and search.

Features

This design demonstrates several key features:

  • Dark/Light theme switching with FOUC prevention
  • 3-column desktop layout (file tree | content | TOC)
  • Responsive breakpoints for tablet and mobile
  • Text wrapping: paragraphs wrap, code/tables scroll
  • Mobile drawer navigation
  • Line numbers with user-select: none
